# lowRISC Style Guides

[Permalink: lowRISC Style Guides](https://github.com/lowRISC/style-guides#lowrisc-style-guides)

This repository contains style guides curated by lowRISC for use in our
code and documentation.

- [SystemVerilog style guide](https://github.com/lowRISC/style-guides/blob/master/VerilogCodingStyle.md)
- [Coding Style Guide for Design Verification](https://github.com/lowRISC/style-guides/blob/master/DVCodingStyle.md)

We invite issues and pull requests to add clarity to these guides.
