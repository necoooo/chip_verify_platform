## 2 简介

UART 模块实现标准异步串行通信功能，支持全双工收发。波特率、数据格式通过 AHB 总线寄存器配置，适用于芯片验证教学中的基本通信场景。

---

## 3 特点

- 全双工异步收发
- 默认波特率 115200 bps，AHB 可配
- 数据格式：8位数据、1位停止、无校验（默认）
- 波特率分频器可配置，适应不同系统时钟频率
- AHB-Lite 从机接口，支持寄存器读写
- 状态寄存器反映收发状态

---

## 4 基本原理

UART 收发基于波特率时钟（由系统时钟 hclk 分频产生）进行比特采样。发送端将并行数据通过移位寄存器串行输出（LSB first）；接收端对 RX 引脚进行16倍过采样，在每位中点采样，实现可靠的异步接收。

## 7 时钟和复位

- **时钟域**: hclk 域
- **复位**: 复位后所有寄存器恢复默认值，收发状态机回到 IDLE

---

## 8 功能描述

### 8.1 波特率发生器

波特率时钟 = hclk / (BAUD_DIV + 1)

115200 bps @ 50MHz → BAUD_DIV = 50,000,000 / 115200 - 1 ≈ 433

### 8.2 数据发送

1. 软件写 TXD 寄存器
2. 硬件自动启动发送序列：起始位(0) → D0...D7(LSB first) → 停止位(1)
3. 发送完成后置位 TX_DONE 标志，可选产生中断

### 8.3 数据接收

1. 硬件检测 RX 引脚下降沿（起始位）
2. 在每位中点采样（16倍过采样）
3. 接收8位数据 + 停止位
4. 接收完成后置位 RX_VALID 标志，数据存入 RXD 寄存器

## 9 寄存器描述

### 9.1 寄存器列表

| 偏移地址 | 寄存器名 | 属性 | 复位值 | 说明 |
|----------|----------|------|--------|------|
| 0x00 | UART_CTRL | RW | 0x0000_0003 | 控制寄存器 |
| 0x04 | UART_BAUD | RW | 0x0000_01B1 | 波特率分频寄存器（默认433） |
| 0x08 | UART_STATUS | RO | 0x0000_0000 | 状态寄存器 |
| 0x0C | UART_TXD | WO | 0x0000_0000 | 发送数据寄存器 |
| 0x10 | UART_RXD | RO | 0x0000_0000 | 接收数据寄存器 |

### 9.2 UART_CTRL（0x00）

| 位 | 属性 | 复位值 | 说明 |
|----|------|--------|------|
| [0] | RW | 1'b1 | TX_EN：发送使能 |
| [1] | RW | 1'b1 | RX_EN：接收使能 |
| [2] | RW | 1'b0 | PARITY_EN：校验使能（预留） |
| [4:3] | RW | 2'b00 | PARITY_SEL：校验选择（预留） |
| [31:5] | RSVD | 27'h0 | 保留 |

### 9.3 UART_BAUD（0x04）

| 位 | 属性 | 复位值 | 说明 |
|----|------|--------|------|
| [15:0] | RW | 16'd433 | BAUD_DIV = hclk / 波特率 - 1（433→115200@50MHz） |
| [31:16] | RSVD | 16'h0 | 保留 |

### 9.4 UART_STATUS（0x08）

| 位 | 属性 | 复位值 | 说明 |
|----|------|--------|------|
| [0] | RO | 1'b0 | TX_BUSY：发送忙 |
| [1] | RO | 1'b0 | TX_DONE：发送完成（读后自动清零） |
| [2] | RO | 1'b0 | RX_VALID：接收数据有效（读后自动清零） |
| [3] | RO | 1'b0 | RX_OVERFLOW：接收溢出 |
| [4] | RO | 1'b0 | FRAME_ERR：帧错误 |
| [31:5] | RSVD | 27'h0 | 保留 |

### 9.5 UART_TXD（0x0C）

| 位 | 属性 | 复位值 | 说明 |
|----|------|--------|------|
| [7:0] | WO | 8'h00 | TX_DATA：发送数据 |
| [31:8] | RSVD | 24'h0 | 保留 |

### 9.6 UART_RXD（0x10）

| 位 | 属性 | 复位值 | 说明 |
|----|------|--------|------|
| [7:0] | RO | 8'h00 | RX_DATA：接收数据 |
| [31:8] | RSVD | 24'h0 | 保留 |

## 10 软件配置流程

```
UART初始化（115200bps @ 50MHz）：
  1. 写 UART_CTRL = 0x0   // 关闭收发使能
  2. 写 UART_BAUD = 433   // 波特率分频
  3. 写 UART_CTRL = 0x3   // 使能收发

发送数据：
  1. 读 UART_STATUS，确认 TX_BUSY=0
  2. 写 UART_TXD = data   // 启动发送
  3. 等待 TX_DONE=1（轮询或中断）

接收数据：
  1. 等待 RX_VALID=1（轮询或中断）
  2. 读 UART_RXD = data
```
